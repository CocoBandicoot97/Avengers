JAITA54

Vuoi diventare un Avengers in Java?
Questo è il posto giusto per te!

Il corso utilizza una metodologia didattica innovativa, pratica e modellata sulla base delle reali necessità delle più importanti aziende del settore. Rivolto principalmente a giovani in cerca di occupazione tra i 18 e i 29 anni. La passione per i sistemi informatici e per il mondo della programmazione sono un valore aggiunto importante per raggiungere gli obiettivi professionali del corso.

La figura professionale del Junior Java Developer fa parte di un team che progetta, realizza e perfeziona applicazioni, siti e software. Lo Sviluppatore Java è generalmente coinvolto in tutte le aree di sviluppo, design e test del prodotto.

Programma del corso

  • Architettura e ciclo di vita delle applicazioni
  • Applicazioni Java (Java 11 Programming I)
  • Sviluppo lato front-end facendo uso di HTML, CSS, JavaScript, JQuery, Bootstrap
  • Fondamenti di progettazione e sviluppo di basi di dati relazionali tramite linguaggio MySQL
  • Persistenza dei dati tramite JDBC e JPA (Hibernate)
  • Sviluppo Java web tramite Servlet e JSP
  • Linguaggio UML
  • Pattern MVC
  • Metodologia Scrum
  • Soft Skills
